# python-shopping-list
#This project is a simple shopping list manager built in Python.

# Goals 
- Ask the user how many items they want to buy
- Collect item names and prices with input validation
- Support early exit by typing `"exit"` as item name
- Store items and prices in a dictionary
- Calculate the total cost using a `while` loop - Print all items with their price using a `for` loop
- Skip zero-priced items using `continue`
